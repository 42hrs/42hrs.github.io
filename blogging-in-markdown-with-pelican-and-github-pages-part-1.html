Python and virtualenv

Installing Pelican and Markdown

$ mkvirtualenv pelican
$ workon pelican
$ pip install pelican Markdown ghp-import
$ deactivate

Create a GitHub repo

$ git clone

Kickstart a blog with Pelican

Write your first article

$ touch output/my-first-article.md

Every article in pelican need some metadata like title, date, tags etc. Copy and paste the following content into the file you just created.

output/my-first-article.md

Title: This is my first article
Date: 2014-12-17 20:00
Tags: life, travel
Slug: this-is-my-first-article
Author: John Doe
Summary: Summary of the article
